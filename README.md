# Coffee Shop

App listing coffess and beans to order.

## Installation

Install with yarn

```bash
  yarn
```

Now you need to install the app inside the phone because we are using expo-dev-client

```bash
  yarn ios
```

or

```bash
  expo run:ios
```

## Run Locally

To run the app after the install

```bash
  yarn start
```

## Features

- Immer
- Rive
- Zustand
- Expo-dev-client
- Blur
